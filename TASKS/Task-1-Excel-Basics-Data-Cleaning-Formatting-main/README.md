📄 Data Cleaning Report

Project: Netflix Movies and TV Shows – Data Cleaning Using Excel

1. Introduction

The objective of this task was to clean and standardize a real-world dataset using spreadsheet tools, simulating the responsibilities of a data analyst or intern. I selected the Netflix Movies and TV Shows dataset and performed data cleaning operations to make the dataset ready for analysis.

2. Dataset Description

The dataset contains information about Netflix movies and TV shows, including attributes such as title, director, cast, country, release year, rating, and description. Like most real-world datasets, it contained inconsistencies such as missing values, duplicate records, and formatting issues.

3. Tools Used

Microsoft Excel – primary tool for data cleaning and formatting

GitHub – version control and project documentation

AI assistance (ChatGPT) – used for guidance, clarification of concepts, and best-practice recommendations during the cleaning process

4. Data Cleaning Steps Performed

The following steps were applied to clean and standardize the dataset:

Opened the dataset in Excel and ensured proper column separation and headers

Used Freeze Panes and Filters to efficiently explore the dataset

Identified missing values using filters and conditional formatting

Made informed decisions to leave certain missing values (e.g., cast, director) unchanged to maintain data integrity

Removed duplicate records using a primary key (show_id) after creating a backup copy

Standardized text fields using Excel functions such as:

TRIM() to remove extra spaces

PROPER() for consistent capitalization

UPPER() for categorical values like ratings

Validated and corrected date formats to ensure consistency

Created a separate Cleaned_Data version to maintain a clear separation between raw and processed data

Added a Data_Quality_Notes column to document observations and decisions made during cleaning

5. Use of AI Assistance

AI tools were used as a learning aid throughout the task. ChatGPT assisted by:

Explaining Excel functions such as TRIM, PROPER, and UPPER

Clarifying best practices for handling missing values and duplicates

Providing step-by-step guidance when resolving formula and formatting issues

All cleaning actions were performed manually by me in Excel, with AI used only for conceptual guidance and troubleshooting.

7. Conclusion

This task helped me understand how real-world datasets are cleaned and prepared for analysis. Through this project, I gained hands-on experience with data validation, text standardization, ethical handling of missing values, and proper documentation practices used in professional data workflows.
